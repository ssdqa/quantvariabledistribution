# *Single Site, Anomaly Detection, Cross-Sectional*

*Single Site, Anomaly Detection, Cross-Sectional*

## Usage

``` r
qvd_ss_anom_cs(process_output)
```

## Arguments

- process_output:

  the output of the `qvd_process` function

## Value

a bar plot displaying the proportion of values falling the user-selected
number of SD away from the mean lower and upper outliers are separated
by the color of the bar
